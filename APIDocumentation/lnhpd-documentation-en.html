Parameters

There are three parameters in the request: language, id and format type.

Request parameters
Parameter Meaning Values Required? Notes
lnhpd/api/ingredient/?id= Returns ingredients by id.
    ##### - ID of the ingredients
 Yes
lnhpd/api/ingredient/?lang=en Changes language
  • en - English
  • fr - French
 No, but defaults to English if not specified Determines whether the response is in English or French.
lnhpd/api/ingredient/?type=json Changes the format of result.
  • json - Json
  • xml - Xml
 No, but defaults to Json if not specified Determines whether the result will be of type Json or XML.

Ingredients can be accessed via these URIs:

  • https://api.hres.ca/lnhpd/api/ingredient/?lang=en&type=json
  • https://api.hres.ca/lnhpd/api/ingredient/?lang=fr&type=json

Response format

The response is in JSON, and consists of a result object that contains one objects:

Each ingredient object consists of the following values:

Ingredient object values
Key Value

ingredient_name The .....

matrix_type_code The .....
quantity_list The .....

jQuery example

Example for a function of ingredient to an element with id "#responses", given an ID and language selection:

function getIngredient(id, lang) {
	var base = 'https://imsd.hres.ca';
	var uri = base + '/lnhpd/api/ingredient/?lang=' + lang + '&id=' + id;
	$.ajax({
		url: uri,
		type: 'GET',
		Accept: "application/json",
		dataType: 'json',
		success: function(data) {
			console.log(data.ingredient_name);
			var frag = document.createDocumentFragment();
			var h2 = document.createElement("h2");
			var IngredientName = document.createTextNode(data.ingredient_name);
			var p = document.createElement("p");
			var text = document.createTextNode(data.panels[0].text);
			h2.appendChild(IngredientName);
			p.appendChild(text);
			frag.appendChild(h2);
			frag.appendChild(p);
			$("#responses")[0].appendChild(frag);
		},
		error: function(error) {},
	});
	return;
};
