Introduction

Clinical Trials Database, is providing to the public a listing of specific information relating to phase I, II and III clinical trials in patients. The database is managed by Health Canada and provides a source of information about Canadian clinical trials involving human pharmaceutical and biological drugs.

This Application Programming Interface (API) allows developers to access that information in JSON format for reuse in their own applications.

The base URI for the Notice of Compliance Database  https://api.hres.ca/cta/api/. Any requests are made relative to this URI.

Parameters

Request parameters
Parameter Meaning Values Required? Notes
cta/api/manufacturer/?id= Returns manufacturer by id.
    ##### - ID of the manufacturer
Yes
cta/api/manufacturer/?lang=en Changes language
  • en - English
  • fr - French
 No, but defaults to English if not specified Determines whether the response is in English or French.
cta/api/manufacturer/?type=json Changes the format of result.
  • json - Json
  • xml - Xml
 No, but defaults to Json if not specified Determines whether the result will be of type Json or XML.

Brands can be accessed via these URIs:

  • https://api.hres.ca/manufacturer/api/brand/?lang=en&type=json
  • https://api.hres.ca/manufacturer/api/brand/?lang=fr&type=json

Response format

The response is in JSON, and consists of a result object that contains one objects:

Each regulatory decision object consists of the following values:

Manufacturer object values
Key Value
manufacturer_id The ...
manufacturer_name The .....

Sample response

{"manufacturer_id":1118,"manufacturer_name":"ST JOSEPHS HEALTH CENTRE LONDON"}

jQuery example

Example for a function of manufacturer to an element with id "#responses", given an ID and language selection:

function getManufacturer(id, lang) {
                                var base = 'https://imsd.hres.ca';
                                var uri = base + '/dpd/api/manufacturer/?lang=' + lang + '&id=' + id;
                                $.ajax({
                                    url:uri,
                                    type:'GET',
                                    Accept:"application/json",
                                    dataType: 'json',
                                    success:function(data){
                                        console.log(data.manufacturer_name);
                                        var frag = document.createDocumentFragment();
                                        var h2 = document.createElement("h2");
                                        var Manufacturer = document.createTextNode(data.manufacturer_name);
                                        var p = document.createElement("p");
                                        var text = document.createTextNode(data.panels[0].text);
                                        h2.appendChild(Manufacturer);
                                        p.appendChild(text);
                                        frag.appendChild(h2);
                                        frag.appendChild(p);
                                        $("#responses")[0].appendChild(frag);
                                    },
                                    error:function(error){
                                    },
                                });
                                return;
                            };
