Active Ingredient

TODO put description of active ingredient.....

Paramètres

Il y a trois paramètres dans la requête: language, id and format type.

Request Paramètres
Paramètre Définition Valeurs Requis? Notes
dhpr1/api/activeingredient/?id= Returns active ingredients by id.
    ##### - ID of the active ingredients
 Yes
dhpr1/api/activeingredient/?lang=en Change la langue.
  • en - Anglais
  • fr - Français
 Non, mais l'anglais est la langue par défaut si elle n'est pas spécifiée Détermine si la réponse est affichée en anglais ou en français.
dhpr1/api/activeingredient/?type=json Modifie le résultat du format.
  • json - Json
  • xml - Xml
 Non, mais l'anglais est la langue par défaut si elle n'est pas spécifiée Détermine si la réponse est de type Json ou XML.

Active Ingredients can be accessed via these URIs:

  • https://api.hres.ca/dhpr1/api/activeingredient/?lang=en&type=json
  • https://api.hres.ca/dhpr1/api/activeingredient/?lang=fr&type=json

Format de réponse

The response is in JSON, and consists of a result object that contains one objects:

Each active ingredient object consists of the following values:

Active ingredient object Valeurs
Key Value
id The .....
pharm_form_id The .....
ingredient_prefix The .....
ingredient The .....
ingredient_supplied_ind The .....
strength The .....
strength_unit The .....
strength_type The .....
dosage_value The .....
dosage_unit The .....
notes The .....

Exemple de réponse

{"id":1,"pharm_form_id":1,"ingredient_prefix":"","ingredient":"DESOGESTREL","ingredient_supplied_ind":"I","strength":"0.15","strength_unit":"MG","strength_type":"","dosage_value":"","dosage_unit":"","notes":"HISTORIC FREEFORM TEXT: DESOGESTREL"}

Exemple de jQuery

Example for a function of active ingredient to an element with id "#responses", given an ID and language selection:

function getActiveIngredient(id, lang) {
    var base = 'https://imsd.hres.ca';
    var uri = base + '/dhpr1/api/activeingredient/?lang=' + lang + '&id=' + id;
    $.ajax({
        url:uri,
        type:'GET',
        Accept:"application/json",
        dataType: 'json',
        success:function(data){
            console.log(data.ingredient);
            var frag = document.createDocumentFragment();
            var h2 = document.createElement("h2");
            var ActiveIngredientName = document.createTextNode(data.ingredient);
            var p = document.createElement("p");
            var text = document.createTextNode(data.panels[0].text);
            h2.appendChild(ActiveIngredientName);
            p.appendChild(text);
            frag.appendChild(h2);
            frag.appendChild(p);
            $("#responses")[0].appendChild(frag);
        },
        error:function(error){
        },
    });
    return;
};
